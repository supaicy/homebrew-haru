# homebrew-haru

Homebrew cask tap for [haru](https://github.com/supaicy/haru) — TickTick 스타일 일정관리 macOS 앱.

## 설치

```bash
brew install --cask supaicy/haru/haru
```

## 업데이트

```bash
brew upgrade --cask haru
```

## 제거

```bash
brew uninstall --cask haru
```

## 요구사항

- macOS 11 (Big Sur) 이상
- Apple Silicon (arm64)
